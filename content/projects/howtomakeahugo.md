### Aim
Try to build a hugo website to log everything happened in GigaScience.  

### Description
The website will contain:
1. About
2. Contact
3. Posts
    - Mainly about the learning progress
4. Projects  
    - sharing (FYI only)
5. Weekly plan 
    - Learning plan
    - Todo list  
    - What have been finished/achieved/completed
